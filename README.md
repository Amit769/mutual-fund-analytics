# 📊 Mutual Fund Analytics Dashboard

A comprehensive end-to-end Mutual Fund Analytics project developed during the **Bluestock Fintech Internship**. The project demonstrates data engineering, exploratory data analysis, financial performance analytics, and interactive business intelligence dashboards using Python, SQL, and Power BI.

---

## 📌 Project Overview

This project analyzes mutual fund industry data to generate actionable insights on fund performance, investor behavior, SIP trends, portfolio analytics, and industry growth.

The workflow covers the complete analytics pipeline:

- Data Ingestion
- Data Cleaning & Validation
- SQL Analytics
- Exploratory Data Analysis (EDA)
- Financial Performance Analytics
- Interactive Power BI Dashboard

# 📈 Key Insights

- Industry AUM showed consistent growth between 2022–2025.
- SIP inflows continued to increase across the observed period.
- Large Cap and Flexi Cap funds attracted the highest inflows.
- Risk-adjusted performance varied significantly across fund categories.
- Investor transaction patterns differed across states and age groups.
